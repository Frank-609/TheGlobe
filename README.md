# Geography Quiz Game (C++)

A console-based geography quiz game developed as part of a Programming I Lab university course.

### Features
- Quiz players on countries, capitals, and fun geography facts
- Input is case-insensitive (A/B/C/D/E) in menus
- Full country name input is grammatically checked
- Designed for students, geography enthusiasts, and anyone wanting to train memory

### How to Run
1. Make sure you have a **C++ compiler** installed (e.g., g++).
2. Compile the program:
   ```bash
   g++ main.cpp -o GeographyQuizGame
